# dimuse
Discriminating Music Sequences Methode
Steps in reproducing the results of the research:
1. Install Orange Data Mining (https://orangedatamining.com/download/)
2. Create a Signals Folder for all the files (default folder saved in the .ows files is C:\SemnaleR\) and unzip the files SemnaleR.zip
3. Download all .csv and .ows files in the Signals Folder
4. Run Orange Data Mining and load each .ows file.
5. Check that each of the _CSV File Import_ or _File_ components has the correct path to the specific .csv file (if the default Signals Folder is created, is should work directly)
6. To view each plot, double click on each of the _Scatter Plot_ components.
7. List of tables in DiMuSe MDPI paper:
8. Table 3: MDPI PcaOrangePlot-GeneratedSignals.ows with Dimuse-GeneratedSignal-Mar25-All_MatlabPCA.csv as input
9. Table 12, Table 15, Table 16: MDPI PcaOrangePlot-BaseSet.ows with baseSet-18Mar_MatlabPCA.csv, baseSet_plus_6-18Mar_MatlabPCA.csv, baseSet_minus_6-18Mar_MatlabPCA.csv.
10. Table 17, Table 18: MDPI Dimuse - K+M Base Projection.ows with MDPI Base_6.ian.26_MatlabPCA+projected K-M.xlsx / Sheet: Base + Projected K+M
11. Table 20: MDPI SeccondSet Jan2026.ows with MDPI Weightless Projections.xlsx.
